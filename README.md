# Plasma Calculator

Plasma Calculator is an open-source project within <a href="https://github.com/PlasmaPy/PlasmaPy">PlasmaPy</a>. It is a website that allows any user to quickly calculate any of the fundamental <a href="https://docs.plasmapy.org/en/stable/formulary/parameters.html">Plasma Parameters</a> by entering the required values and their units.  

## Requirements

* Python 3.7+
* All the required packages are listed in the **requirements.txt** file. Everything in the file can be installed at once in the Terminal (macOS) using the command: <br>
`pip install -r requirements.txt` <br> **Note:** This is not required if the "Usage" instructions below are followed

## Usage

* Download the code
* Navigate inside the "Calculator" directory within the project folder in Terminal
* Activate the virtual environment included in the project (macOS): <br>
$ `source env/bin/activate`
* Run the command: <br>
$ `flask run`
* Cmd + Click on the URL provided
* Enjoy the website!
