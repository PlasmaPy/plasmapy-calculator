# Plasma Calculator

Plasma Calculator is an open-source project within <a href="https://github.com/PlasmaPy/PlasmaPy">PlasmaPy</a>. It is a website that allows any user to quickly calculate any of the fundamental <a href="https://docs.plasmapy.org/en/stable/formulary/parameters.html">Plasma Parameters</a> by entering the required values and their units.  

## Requirements

* Python 3.7+
* All the required packages and libraries are listed in the **requirements.txt** file. 

## Usage

* Download the code

* Navigate inside the "Calculator" directory within the project folder in Terminal

* Create a virtual environment: <br>
$ `python3 -m venv env` 

* Activate the virtual environment (macOS): <br>
$ `source env/bin/activate`

* Install all the required packages in the **requirements.txt** file: <br>
$ `pip install -r requirements.txt`

* Run the app: <br>
$ `flask run`

* Cmd + Click on the URL provided

* Enjoy the website!
